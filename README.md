# Gemini Resizer

Gemini Resizer is a small Chrome extension that adjusts the conversation width on Gemini and can optionally align the input box to the same reading column.

## What it does

- Resizes the chat area with a simple popup slider.
- Optionally keeps the input box aligned with the conversation width.
- Saves your settings locally in the browser.

## Files

- `content.js` injects the width controls into Gemini.
- `popup.html` and `popup.js` provide the extension UI.
- `manifest.json` wires the extension together.

## Install

1. Open Chrome and go to `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked and select this folder.

## Usage

Open Gemini, click the extension icon, and adjust the width slider. Use the toggle if you want the input box to follow the same width.
